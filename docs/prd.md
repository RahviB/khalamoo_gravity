Target Language: French (fr-FR)
Region: France (Specific focus: Béarn, Nouvelle-Aquitaine)
1. Executive Summary & Localization
KHALAMOO requires a Headless E-commerce platform. The primary goal is to sell premium hemp products while educating the user on the "Soil to Soul" journey initiated in Lasseube & Gan.
Primary Language: French. All UI elements (Buttons, Errors, Checkout) must be culturally adapted (e.g., "Ajouter au panier" not "Add to cart").
Tone of Voice: "Authentique, Éducatif, Bienveillant, Local."
2. Technical Stack (Refined)
Frontend: Next.js 14 (App Router).
Backend: WordPress (Headless) + WooCommerce.
Locale Handling: next-i18next (Setup for French default, structure ready for future English/Spanish).
Payment Gateway: Stripe (configured for France preferences: CB, Apple Pay) + Option for Klarna (Pay in 3) if basket > €100.
Shipping: Integration with Boxtal or Sendcloud API to offer:
Colissimo (Home delivery).
Mondial Relay (Pickup points - highly popular in France).
